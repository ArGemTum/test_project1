# Заголовок проекта

Данный проект - приложение (графический интерфейс) с подключенной к нему базой данных.

## Начало работы

С помощью git bash на сервер были загружены 
необходимые для работы приложения файлы: 
main.py, captcha.py, MainMenu.py, DataBaseView.py, 
папка files_for_demoexam, в которой располагаются фотографии и иконки.
Кроме того, на сервер были загружены файлы типа .csv, 
где хранятся таблицы для базы данных.

### Необходимые условия

Для того, чтобы установить ПО, необходимо 
загрузить отправленные файлы к себе на компьютер 
и расположить их в среде разработки для python,
например PyCharm. Далее необходимо импортировать в проект 
библиотеку PIL для создания проверки на робота CAPTCHA,
библиотеку PyQt5 для графического интерфейса
и библиотеку psycopg2-binary, psycopg2 для взаимодействия с базой данных.



```
В среде PyCharm можно добавлять библиотеки 
через вкладку "File -> Settings". Далее "Project -> Project Interpreter".
Нажмите на "+" в правом верхнем углу окна, чтобы добавить новую библиотеку.
В появившемся окне выберите библиотеку (PIL, PyQt5, psycopg2-binary, psycopg2) и нажмите на кнопку "Install Package".

Однако также можно добавлять библиотеки через терминал:
pip install psycopg2-binary
pip install PyQt5
pip install pillow

```

### Установка

После добавления библиотек в среде разработки 
необходимо запустить файл main.py, который запустит
авторизационное окно, где необходимо ввести логин и пароль:
логин и пароль для проверки установлены жестко (login: "123", password: "123").
Далее необходимо выполнить проверку на робота CAPTCHA
(на экране появляется код из 7 перечеркнутых символов,
для того чтобы перейти в главное меню приложения,
пользователю нужно верно ввести эти символы
и нажать кнопку войти).

Когда пользователь вводит логин, пароль и CAPTCHA верно,
перед ним открывается графический интерфейс приложения.

В появившемся окне (Главное меню) пользователь 
может перейти к подключению и просмотру базы данных, 
нажав на кнопку: "База данных"

В новом окне (Просмотр базы данных) пользователь 
может в выпадающем списке выбрать нужную ему таблицу 
базы данных и затем нажать на кнопку "Подключиться к базе". 
Чтобы посмотреть другую таблицу, пользователю необходимо 
сначала выбрать таблицу в выпадающем списке,
а затем снова нажать на кнопку "Подключиться к базе". 
После завершения работы, пользователь может вернуться в главное меню, 
нажав на кнопку "Вернуться в меню". 
Окно (Просмотр базы данных) закроется и пользователь вернется в главное меню. 

Далее пользователь может закончить работу с приложением, нажав на кнопку "Выход".
